About Fake News Detection

This AI powered Fake News Detection system helps users identify whether a news article is Real or Fake using Machine Learning.

🎯 Project Objective

The main objective of this project is to detect fake news using Natural Language Processing and Machine Learning algorithms. It helps prevent misinformation and promotes authentic news consumption.

⚙️ Technologies Used

  • Python
  • Flask
  • Machine Learning
  • Scikit-learn
  • Pandas
  • Bootstrap

🧠 How It Works

  1. User enters news text
  2. Text converted into TF-IDF vector
  3. Model predicts Fake or Real
  4. Confidence score shown
